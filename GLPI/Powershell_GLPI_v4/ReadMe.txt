POWERSHELL_GLPI

This module is made for mass deployment of GLPI entities
Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.
Prerequisites

You need powerhsell ISE v 5.1

Installing

- To install this modul unzip the archive
- When its done you have 2 XML file, 1 ReadMe and 1 folder Name Powershell_GLPI
- Open Powerhsell ISE
- Type the following command : $env:PSModulePath
- Take the second link (C:\Program Files\WindowsPowerShell\Modules)
- Copy paste the folder Powershell_GLPI(that is in the zip) to the link
- When its done open Script-ajout-entité.ps1
- Configure it and launch it
